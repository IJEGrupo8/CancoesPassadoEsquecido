## Para compilar (Linux):

Entre na pasta do projeto e crie uma pasta chamada build.
```shell

  cd CancoesPassadoEsquecido/
  mkdir build

```
Após isso rode o cmake e o make.
```shell
  cmake ..
  make
```
## Para rodar (Linux):

```shell
  ./cancoes
```
